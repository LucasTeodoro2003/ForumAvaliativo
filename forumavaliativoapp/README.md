# App de Cadastro de Produtos

Este é o frontend de um aplicativo móvel desenvolvido para a disciplina Desenvolvimento para Dispositivos Móveis.

## 🛠️ Tecnologias Utilizadas
* React
* NextJS
* Supabase
* Better Auth
* Prisma

## ⚙️ Como rodar o projeto
1. Clone este repositório: `git clone https://github.com/LucasTeodoro2003/ForumAvaliativo.git`
2. Instale as dependências: `npm i`
3. Inicie o app: `npm run dev`
